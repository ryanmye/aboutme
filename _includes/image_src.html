{%- comment -%} Pluggable image URL resolver. All album/gallery image URLs go through here so the delivery backend can be swapped via site.images.source in _config.yml. Inputs: src - repo-relative path, e.g. "/assets/images/posts/foo.png" variant - "thumb" | "med" | "original" Output: a single URL string; callers should wrap in a capture tag and pipe through strip. Backends: local (default) - uses _data/image_meta.yml to map to pre-generated variants cloudflare_r2 - serves variants from a public R2 custom domain Falls back to the original src whenever a variant cannot be resolved, so the page always renders something even if the manifest is stale. {%- endcomment -%} {%- assign _src = include.src -%} {%- assign _variant = include.variant | default: "original" -%} {%- assign _src_rel = _src | remove_first: '/' -%} {%- assign _meta_key = _src_rel | remove_first: 'assets/images/' -%} {%- assign _meta = site.data.image_meta[_meta_key] -%} {%- assign _source = site.images.source | default: "local" -%} {%- if _source == "cloudflare_r2" and site.images.cloudflare.r2_public_base_url != "" -%} {%- assign _r2_key = _meta_key -%} {%- if _variant != "original" and _meta and _meta[_variant] and _meta[_variant].src -%} {%- assign _r2_key = _meta[_variant].src -%} {%- endif -%} {{ site.images.cloudflare.r2_public_base_url | append: '/' | append: _r2_key }} {%- else -%} {%- if _variant != "original" and _meta and _meta[_variant] and _meta[_variant].src -%} {%- assign _variant_rel = _meta[_variant].src | prepend: '/assets/images/' -%} {{ _variant_rel | relative_url }} {%- else -%} {{ _src | relative_url }} {%- endif -%} {%- endif -%}
